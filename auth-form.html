Окно входа/авторизации(Общий компонент)

Для смены состояний полей ввода, необходимо присвоить элементуds-inputs-component:
1) классis-error при ошибке ввода;
2) класс is-hidden-snowflake для скрытия красной снежинки;
3) класс is-hidden-info для скрытия подсказок снизу;
4) класс .is-unavailable для неактивного состояния поля;

Если введены неправельные данные, появляется текст с ошибкой над кнопкой. Для этого, необходимо присвоить класс is-errorэлементуauth-form__form.

Для отображения активной кнопки, удалить класс .ds-btn_disabled и добавить .ds-btn_inverse.

Для активации вкладки, необходимо присвоить класс .is-active элементу .auth-form__left-tab или .auth-form__right-tab

Для стилизации логина при входе на сайт, необходимо присвоить класс .input-login__signIn елементу ds-inputs-form, а также добавить лейбл Логин /email /телефон - как в примере
