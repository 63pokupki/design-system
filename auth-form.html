Окно входа/авторизации

У окна два состояния: 1) окно входа(по дефолту), 2) окно регистрации. Для отображения окна регистрации, необходимо присвоить модификатор auth-form_sign-up элементу auth-form.
Для смены состояний полей ввода, необходимо присвоить элементуds-inputs-component:
1) класс is-typing при вводе текста;
2) классis-error при ошибке ввода;
3) класс is-hidden-snowflake для скрытия красной снежинки;
4) класс is-hidden-info для скрытия подсказок снизу;
5) класс .is-unavailable для неактивного состояния поля;
