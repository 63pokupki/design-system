Контейнер для изображений

Контейнер используется для быстрой вставки изображения любого размера.
Опции: _extra-large - для очень больших экранов, _large - для больших, _middle - для средних, _small - для маленьких, _resp - резиновый
Для включения опции, необходимо элементу img-container прописать модификатор. Например: img-container_large
Для вертикальных изображений рекомендуется использовать универсальный модификатор _resp
