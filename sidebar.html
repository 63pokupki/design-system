Боковое меню (контейнер)

Контейнер для бокового меню. Содержит затемняющий фон, кнопку закрытия и саму панель

По умолчанию выезжает слева. Для выезжания справа добавить класс модификатор .sidebar_right

Для активации меню добавить класс модификатор .sidebar_is-active

Для отображения только на сенсорных устройствах (до 1024px) добавить служебный класс .touch-only

В .sidebar__body находится контент панели - внутрь класть все содержимое. Максимальная ширина 90vw

Для отображения уменьшить ширину браузера
