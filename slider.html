Слайдер

Опция: при добавлении класса .no-btn скрываются кнопки. .no-l-btn - левая кнопка, .no-r-btn правая кнопка.
Для этого необходимо добавить их элементу slider
Смещение элементов слайдера происходит при изменении свойства transform: translateX(0) у элемента slider-wrap
