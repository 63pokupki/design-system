Как выставлять значения для визуального отображения ценового диапазона:

Для ползунков:

Для задания значения левого и правого ползунков надо менять их inline значение left.
0% от максимальной суммы фильтра будет равен left: 0%, а 100% от максимальной суммы фильтра будет равен left: 100%

Для активной полоски на заднем фоне

Левое положение ставится равным положению ползунка минимума. Правое ставится по inline значению width (ширина)
Ширина высчитывается как: Значение ползунка максимума - Значение ползунка минимума (смотреть в примерах)
